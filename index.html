Request and data flow

  1. Ingest: register sources, normalize text, chunk documents, and persist metadata.
  2. Index: create embeddings through provider seams and store workspace-scoped vectors.
  3. Retrieve: run scoped retrieval with diagnostics and ranking controls.
  4. Generate: return cited answers with confidence and fallback behavior.
  5. Operate: monitor health, metrics, evaluations, and hardening gaps.
